Shaping a glowing circle

Last lesson you made a distance field — dark at a point, brighter outward. Now you turn distance into form: a crisp, glowing circle. This is the move that scales all the way up to a raymarched sphere.

Your win this lesson Define a shape as a signed distance (length(uv) - r) and paint it with a soft, glowing edge using smoothstep and mix.

Center the coordinates first

For shapes we switch to the centered, aspect-correct UV — origin in the middle, divided by height so circles stay round (Lesson 01 ended on exactly this line):

vec2 uv = (fragCoord - 0.5*iResolution.xy) / iResolution.y;  // 0,0 = screen center

Distance → shape

The distance from the center is length(uv). Subtract a radius and you get the signed distance to a circle:

float d = length(uv) - r;   // d < 0 inside, d = 0 on the edge, d > 0 outside

That single sign tells you inside from outside. To draw it, you map d to a brightness. A hard cut is step(0.0, -d) (1 inside, 0 outside) — but it aliases into jaggies. The artist's tool is smoothstep, which fades smoothly across a thin band, giving a clean anti-aliased edge — or, widened, a glow:

float circle = 1.0 - smoothstep(0.0, aa, d);   // aa = edge softness

Then mix blends two colors by that 0→1 mask: mix(background, fill, circle).

Why this matters for raymarching length(uv) - r in 2D becomes length(p) - r in 3D — the exact SDF of the sphere you'll raymarch in Lesson 04. smoothstep and mix stay your edge-and-color tools the whole way up. You're learning the real vocabulary now, just in 2D.

Play — sculpt the glow

No code: drag the sliders. radius sizes it, edge softens the rim, glow adds a halo. This warm, glowing look is pure creative-art territory.

Now you — build the circle, graded live

The starter shows nothing: d = length(uv) has no radius, so the shape is empty. Fix it, then tint it. Hit Check on each — it reads the pixels back and grades you.

Watch for smoothstep(a, b, x) needs a and b different, and goes 0→1 as x moves from a to b. If your shape is inverted, you likely swapped them or forgot the 1.0 -.
