Raymarching for Creative Art · Lesson 10 · Infinite detail

Fractals: the Menger sponge

This is what raymarching does that nothing else can touch: infinite, self-similar detail from a few lines of math. A loop that carves the same hole at ever-smaller scales builds a structure no mesh could ever store.

Your win this lesson Understand and run an iterated SDF — the Menger sponge — and control its recursion depth, the moment procedural detail becomes effectively free.

Self-similarity by iteration

Start with a box. Carve a cross-shaped hole through it (that's the "+1" Menger step). Now do the same carve again on each remaining piece, three times smaller — and again, and again. Each pass multiplies the detail; the SDF stays a dozen lines:

float menger(vec3 p) {
    float d = sdBox(p, vec3(1.0));
    float s = 1.0;
    for (int m = 0; m < ITER; m++) {
        vec3 a = mod(p*s, 2.0) - 1.0;
        s *= 3.0;
        vec3 r = abs(1.0 - 3.0*abs(a));
        float c = (min(min(max(r.x,r.y), max(r.y,r.z)), max(r.z,r.x)) - 1.0) / s;
        d = max(d, c);               // subtract the cross at this scale
    }
    return d;
}

Each iteration adds a finer level of holes. Three or four passes already looks impossibly intricate; the cost is just a few more loop turns per ray step.

Watch for Iterated SDFs are only approximate distances (they slightly over-estimate), so rays can overshoot thin features. If the surface looks eroded, lower the hit threshold or take a fraction of the step (t += d*0.8). More iterations also cost more — there's a real budget here.
