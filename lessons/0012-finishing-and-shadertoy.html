Finishing & shipping to Shadertoy

You have every technique. This lesson is about the last 10% that separates a demo from a piece — materials, polish, and the small ritual of publishing it where people will see it.

Your win this lesson Take a complete scene, give it materials and finishing polish, make it yours, and publish it to Shadertoy — your mission's destination.

Materials: make map return an id

To shade objects differently, return a material id alongside the distance — a vec2 of (distance, id) — and switch on it when shading:

vec2 map(vec3 p) {
    vec2 res = vec2(p.y + 1.0, 0.0);              // ground = id 0
    float blob = length(p) - 0.9;
    if (blob < res.x) res = vec2(blob, 1.0);      // blob = id 1
    return res;
}
// ... later: vec3 base = (id == 0.0) ? floorColor : blobColor;

The finishing polish

  • Gamma — always end with pow(col, vec3(0.4545)). Non-negotiable.
  • Fog — mix(col, sky, 1.0-exp(-t*k)) for depth and to hide the far clip.
  • Vignette — darken the corners a touch; the eye goes to the center.
  • Tone & contrast — a little col = col*col*(3.0-2.0*col) (smoothstep) deepens it.
  • Antialiasing — if you can afford it, average 2×2 sub-pixel rays. Optional; costly.

Your capstone — make it yours, graded live

A complete scene: ground, a smin blob, cosine palette, specular, soft shadow, fog, vignette, gamma — everything from the course, and no custom uniforms, so it pastes straight into Shadertoy. It's static. Your task: bring it to life by orbiting the camera with iTime. Then make it yours — change the palette, the light, the shapes. Check confirms it animates.

Publish it to Shadertoy

  1. Open shadertoy.com/new (sign in to save/publish).
  2. Select all the code in the editor above and paste it over the default mainImage. The iTime, iResolution, and iMouse uniforms are already there.
  3. Bake any slider values into constants. The lab "Play" scenes use custom uniforms (uCov, uShin…) that Shadertoy doesn't have — replace each with the number you liked, e.g. float uShin = 24.0;. (This capstone already has none.)
  4. Hit the ▶ (or Alt+Enter) to compile. Fix any red errors in the log at the bottom.
  5. Give it a name and description, set a license, choose Public or Unlisted, and Publish. Share the link.
