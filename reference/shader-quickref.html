The harness contract

Every lab + Shadertoy gives you these for free:

uniform vec3  iResolution; // viewport px (.xy)
uniform float iTime;       // seconds since start
uniform vec4  iMouse;      // .xy = px while dragging
                           // .z  > 0 while pressed

void mainImage(out vec4 fragColor,   // ← you write
               in  vec2 fragCoord) { // ← pixel pos
    fragColor = vec4(1.0, 0.0, 0.0, 1.0); // red
}

in = given to you · out = you fill it.

UV recipes

// 0..1 across screen (color/texture work)
vec2 uv = fragCoord / iResolution.xy;

// centered, aspect-correct (shapes, SDFs,
// raymarch camera) — 0,0 is screen center
vec2 uv = (fragCoord - 0.5*iResolution.xy)
          / iResolution.y;

Use the centered form for anything geometric — it keeps circles round on wide screens and puts the origin where you think it is.

Vector types & swizzling

vec2 a = vec2(1.0, 2.0);
vec3 c = vec3(1.0);        // (1,1,1)
vec4 p = vec4(c, 1.0);    // build from parts
float x = p.x;  vec2 yz = p.yz;   // swizzle
vec3 bgr = c.bgr;          // reorder freely

Color is just a vector: vec3 = (r,g,b), vec4 = (r,g,b,a), each component 0→1.

GLSL gotchas

  • Type-strict: 1/2 ✗ → write 1.0/2.0. Always put .0 on floats.
  • No implicit int↔float. float(i) to convert a loop counter.
  • Y points up (bottom-left origin), unlike most 2D screen APIs.
  • The lab's status bar shows the exact error line on a typo.

The raymarching loop (the destination)

Where the course is heading — module 05 in ten lines. Each future lesson expands one part:

float map(vec3 p) { return length(p) - 1.0; }     // the scene: a unit sphere as a distance field

float raymarch(vec3 ro, vec3 rd) {                // ro = ray origin (camera), rd = ray direction (per pixel)
    float t = 0.0;
    for (int i = 0; i < 100; i++) {
        float d = map(ro + rd * t);               // distance to nearest surface from here
        if (d < 0.001) return t;                  // hit
        if (t > 50.0)  break;                      // miss — gone too far
        t += d;                                    // safe step: move by the closest distance
    }
    return -1.0;
}

Don't worry if this is opaque now — by the time we reach it, every line will be something you've already built.
